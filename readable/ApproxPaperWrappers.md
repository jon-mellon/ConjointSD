# ApproxPaperWrappers.lean

Lean file: [ConjointSD/ApproxPaperWrappers.lean](../ConjointSD/ApproxPaperWrappers.lean)

This file contains the approximate/misspecification variants of the paper-facing SD wrappers.

What it covers:
- Approximate target SD bounds for block and total scores under `ApproxInvarianceAE`.
- Approximate links to `gStar` via `ApproxWellSpecifiedAE`.
- Two-stage oracle approximation via `ApproxOracleAE`.

These results are not used in the main theorem chain.
