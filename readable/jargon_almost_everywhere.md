# Almost everywhere (a.e.)

Lean entrypoint: [ConjointSD.lean](../ConjointSD.lean)

A statement holds "almost everywhere" under a [distribution](jargon_distribution.md) if it can fail only on a set of outcomes with probability 0. Informally: it is true for all practical purposes under that distribution.
