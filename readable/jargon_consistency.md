# Consistency

Lean entrypoint: [ConjointSD.lean](../ConjointSD.lean)

An [estimator](jargon_estimator.md) is consistent if it [converges](jargon_convergence.md) to the true target as the sample size grows. In these files, this is usually expressed with a convergence statement.
