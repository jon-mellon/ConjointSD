# Normalization

Lean entrypoint: [ConjointSD.lean](../ConjointSD.lean)

Normalization rescales something so it has a standard total, often 1. In weighted [averages](jargon_mean.md), you divide by the total weight so the result is on the same scale as an ordinary average.
